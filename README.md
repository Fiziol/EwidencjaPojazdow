# Projekt: Ewidencja Pojazdów

Zadanie zaliczeniowe wykonane w ramach zajęć, obejmujące techniki programowania desktopowego oraz webowego.

## Zakres projektu:

### ETAP 1 & 2 – Visual Studio & WinForms
- Projektowanie interfejsu w WinForms (C#).
- Obsługa bazy pojazdów (Nr rejestracyjny, Marka, Model, Rok, Data przeglądu, Właściciel).
- Zarządzanie danymi za pomocą listy `BindingList`.
- Wykorzystanie technologii **LINQ** do filtrowania raportu pojazdów z przeglądem na bieżący miesiąc.
- Prezentacja danych w `DataGridView`.

### ETAP 3 & 4 – VS Code & Bootstrap 5
- Budowa responsywnego interfejsu webowego.
- Wykorzystanie **Bootstrap 5** do stylizacji tabeli i formularzy.
- Implementacja okien modalnych (dodawanie/usuwanie pojazdu).
- Walidacja formularzy po stronie klienta za pomocą JavaScript.
- Dynamiczne alerty o statusie operacji.

## Technologie:
- **Języki:** C#, HTML, CSS, JavaScript
- **Frameworki:** .NET WinForms, Bootstrap 5
- **Narzędzia:** Visual Studio 2022, Visual Studio Code, GitHub
